== Description ==

Ce widget affiche l'agenda des sorties de votre ville et de sa région, en lien avec le site [Fest](https://www.fest.fr) : concerts, spectacles, théâtre, soirées, expositions, fêtes, brocantes, vide-greniers, lotos...

Fest vous aide à trouver l'événement qui vous fera passer un bon moment entre amis ou en famille. Du méga concert au loto communal, Fest référence les idées de sorties sur toute la France. [Annoncer un événement](https://www.fest.fr/annoncer.html) est simple et gratuit.

== Installation ==

Décompressez l'archive et placez le dossier `fest` dans le répertoire `/wp-content/plugins/` de votre installation WordPress. Activez le plugin depuis le menu `Extensions` de Wordpress.
